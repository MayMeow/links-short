## Installation

```bash
# clone this repository

# Create data folder & set permissions
mkdir ./data && chown 11235:11235 ./data

# Download docker image & run migration
docker compose pull
docker compose run --rm app php bin/cake.php migrations migrate --no-lock
docker compose up -d
```
